💡 Log Tips
  • Logs are stored in an in-memory ring buffer (last 2,000 lines). They reset on server restart.
  • Use Auto-Refresh to watch logs live as users interact with the platform.
  • Filter by ERROR or CRITICAL first when diagnosing problems.
  • Search by email, venue name, artist name, or endpoint path to trace a specific user's actions.
  • For persistent log storage, set GIGSFILL_LOG_FILE=/var/log/gigsfill.log on your server.
